# 2022年推荐学习和不推荐学习的编程语言
### 选择学习哪门编程语言，首先需要具备以下观念，然后再来确定要学习哪门语言。

 1. 编程语言有很多种，这世上没有完美的编程语言。每门语言都有不少缺陷。
 2. 不同语言的适用范围不同，我们应该根据应用情况来选择合适的编程语言。
 3. 编程语言都在不断地发展变化，语言会不断优化升级，最终也都会被淘汰。
